Vim:
will create file if doesnt exist
i - insert mode
v - visual mode
esc - return to command mode

:wq! - in command mode to write and quit
:q! - in command mode to quit without saving changes
:w - save

:n - jump to line n
w - forward a word
b - back a word
0 - begining of line
* - end of line? (doesnt work for me)???
h j k l - left down up right :)

/word - to search for word
n - for next word
N for previous word

dd - delete line
D - delete from cursor to end of line
yy - copy line
y$ - copy from cursor to end of line
p - paste below
P - paste above
u - undo
ctrl + r - redo

:syntax on - colour syntax
:set number - show line numbers
:set nonumber - hide line numbers
